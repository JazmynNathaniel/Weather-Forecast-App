# Weather Forecast App

A simple weather app that shows current conditions plus hourly and daily forecasts for a searched city.

## Features
- Current weather with emoji and temperature in °C/°F
- 24-hour forecast (3-hour intervals)
- 7-day forecast

## How To Run
1. Open `WEATHER_APP/index.html` in your browser.
2. Enter a city name and submit.

## Notes
- Uses the OpenWeatherMap API in `WEATHER_APP/app.js`.
